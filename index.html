📘 Belajar HTML: Contoh Langsung + Cara Nulis

Halaman ini dirancang responsif untuk HP & PC. Setiap fitur ditampilkan hasilnya diikuti dengan kode HTML yang ditulis. Jadi kamu bisa lihat dan langsung tiru!

🖼️ 1. Gambar

<img src="https://placehold.co/600x300/...?text=Contoh+Gambar" alt="deskripsi">

Gunakan <img> dengan atribut src (URL gambar) dan alt (teks alternatif).

🔗 2. Link / URL

Contoh link: Kunjungi Google (buka tab baru)

<a href="https://www.google.com" target="_blank">Kunjungi Google</a>

✨ 3. Inline Code Highlights

Di dalam paragraf, kamu bisa menulis kode seperti console.log("Halo Dunia") menggunakan tag <code>.

<code>console.log("Halo Dunia")</code>

📦 4. Block Code (Pre + Code)


// Ini contoh block code
function sapa(nama) {
    return `Halo, ${nama}!`;
}
console.log(sapa("Belajar HTML"));
<pre><code> ... kode di sini ... </code></pre>

💬 5. Blockquote

"Ini adalah kutipan panjang. Biasanya digunakan untuk mengambil teks dari sumber lain. Belajar HTML itu menyenangkan!"
— Dari seorang pelajar
<blockquote> ... teks kutipan ... <footer>— sumber</footer></blockquote>

✍️ 6. Format Teks (Tebal, Miring, Underline, dll)

Setiap contoh di bawah menunjukkan hasil yang terlihat dan di bawahnya ada cara penulisan kode.

🔤 Tebal (Strong / B)
Ini teks tebal yang penting
<strong>Ini teks tebal yang penting</strong> atau <b>...</b>
📖 Miring (Em / I)
Teks miring untuk penekanan
<em>Teks miring untuk penekanan</em> atau <i>...</i>
✍️ Underline (Garis Bawah)
Teks ini bergaris bawah
<u>Teks ini bergaris bawah</u>
🟨 Highlight (Mark)
Teks yang disorot seperti stabilo
<mark>Teks yang disorot seperti stabilo</mark>
🔻 Teks Kecil (Small)
Catatan kecil atau disclaimer
<small>Catatan kecil atau disclaimer</small>
❌ Teks Dicoret (Del)
Harga lama Rp 200.000 Harga baru Rp 150.000
<del>Harga lama Rp 200.000</del> Harga baru Rp 150.000
➕ Teks Sisipan (Ins)
Ini teks yang ditambahkan kemudian
<ins>Ini teks yang ditambahkan kemudian</ins>
🔢 Superscript (Pangkat)
Rumus luas: m2 , E = mc2
m<sup>2</sup> dan E = mc<sup>2</sup>
⬇️ Subscript (Subskrip)
Air = H2O , CO2
H<sub>2</sub>O dan CO<sub>2</sub>
🎨 Kombinasi
Tebal, miring, dan garis bawah sekaligus
<strong><em><u>Tebal, miring, dan garis bawah</u></em></strong>

📋 7. Daftar (List)

Unordered List (bullet) pakai <ul> <li>

  • Belajar HTML
  • Belajar CSS
  • Belajar JavaScript
<ul><li>Belajar HTML</li><li>Belajar CSS</li></ul>

Ordered List (angka) pakai <ol> <li>

  1. Langkah 1: Buka editor
  2. Langkah 2: Tulis HTML
  3. Langkah 3: Simpan .html
<ol><li>Langkah 1...</li><li>Langkah 2...</li></ol>
