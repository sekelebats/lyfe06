📚 Belajar Elemen HTML Dasar

Halaman ini berisi contoh gambar, link, kode, blockquote, dan berbagai macam format teks.

🖼️ Gambar

Gunakan tag <img src="url_gambar" alt="deskripsi">.

🔗 URL / Link

Kunjungi Google menggunakan tag <a>.

✨ Inline Code Highlights

Untuk menandai potongan kode di dalam teks, gunakan tag <code>. Contoh: fungsi console.log("Halo") akan mencetak pesan.

📦 Block Code


// Contoh block code dalam HTML/CSS/JS
function sapa(nama) {
    return `Halo, ${nama}!`;
}
console.log(sapa("Dunia"));

Gunakan <pre><code>...</code></pre> untuk menampilkan kode multi-baris.

💬 Blockquote

"Kutipan ini menunjukkan penggunaan blockquote. Biasanya digunakan untuk mengutip pernyataan dari sumber lain."
— Seseorang Bijak

Gunakan tag <blockquote> untuk membuat kutipan panjang.

✍️ Format Teks: Tebal, Miring, Underline, dan Lainnya

Teks tebal (strong) — menggunakan tag <strong> atau <b>.

Teks miring (emphasized) — menggunakan tag <em> atau <i>.

Teks bergaris bawah (underline) — menggunakan tag <u>.

Teks ditandai (highlight) — menggunakan tag <mark>.

Teks kecil (small) — menggunakan tag <small>.

Teks dicoret (deleted) — menggunakan tag <del>.

Teks bergaris bawah sebagai sisipan (inserted) — menggunakan tag <ins>.

Contoh superscript (pangkat) seperti E = mc2 — tag <sup>.

Contoh subscript (bawah) seperti H2O — tag <sub>.

Kombinasi: Tebal, miring, dan bergaris bawah.

📋 Daftar (List)

Daftar tidak berurutan (<ul>):

  • Item pertama
  • Item kedua
  • Item ketiga

Daftar berurutan (<ol>):

  1. Langkah pertama
  2. Langkah kedua
  3. Langkah ketiga

Daftar definisi (<dl>):

HTML
HyperText Markup Language, kerangka halaman web.
CSS
Cascading Style Sheets, untuk tampilan.
JavaScript
Bahasa pemrograman untuk interaktivitas.
